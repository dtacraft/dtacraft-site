Collapse State

Privacy Policy

Effective Date: 27/02/2026

Collapse State (“the Game”) is a premium, offline, single-player game developed and published by DTA Craft.

We respect player privacy and are committed to transparency and minimal data handling.

1. Information We Do Not Collect

Collapse State does not collect, store, transmit, or process any personal data.

We do not collect:

  • Names, email addresses, or contact information
  • Location data
  • Device identifiers
  • Usage analytics or behavioral data
  • Advertising identifiers
  • Payment or financial information

The Game does not require an account or online connection.

2. Game Data

All game data, including:

  • Save files
  • Progress
  • Settings

is stored locally on the player’s device only.

This data:

  • Is never transmitted to any server
  • Cannot be accessed by the developer
  • Can be deleted by uninstalling the Game

3. Third-Party Services

Collapse State does not use:

  • Analytics services
  • Advertising networks
  • Cloud saves
  • Third-party SDKs that collect user data

The Game operates entirely offline after installation.

4. Children’s Privacy

Collapse State does not knowingly collect any personal information from children under the age of 13.

Because no personal data is collected at all, the Game complies with applicable child privacy regulations.

5. Data Security

Since Collapse State does not collect or transmit personal data, there is no risk of personal data exposure through our systems.

6. Changes to This Policy

This Privacy Policy may be updated if the Game’s features change in a way that affects data handling.

Any updates will be reflected on this page.
